## Layer role

This repository is an implementation utility layer.

It should hold:

- cargo tracking bot code and utility logic;
- implementation-facing documentation;
- release surfaces for the bot itself;
- a readable human entry into what the bot does.

It should not become:

- a one-line placeholder shell;
- a donor governance repository;
- a generic ecosystem dump;
- an undocumented bot fragment.
